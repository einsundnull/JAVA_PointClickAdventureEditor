 ✅ Vollständige Orientation-Image-Implementierung

  🎯 Kernfunktionalität

  Wie es funktioniert:
  1. Item mit isFollowingMouse oder isFollowingOnMouseClick aktivieren
  2. Orientation-Images im Editor zuweisen
  3. Maus bewegen → Orientation wird automatisch berechnet
  4. Debug-Log: "Item 'MyItem' Orientation: TopLeft" etc.
  5. Passende Bild wird automatisch gerendert

  📦 Implementierte Komponenten:

  - Sternförmige Berechnung mit 8 Sektoren à 45° (main/Item.java:753-808)
  - Debug-Logging bei Orientierungswechsel (main/Item.java:804-807)
  - Smart-Image-Selection (main/Item.java:817-835)

  for (Item item : currentScene.getItems()) {
      if (item.isFollowingMouse() || item.isFollowingOnMouseClick()) {
          item.updateOrientationBasedOnCursor(e.getX(), e.getY());
      }
  }

  4. ItemOrientationEditor.java - Visual Editor ⭐ NEU!

  Features:
  - ComboBox mit allen Items aus aktueller Scene
  - 10 Drop-Felder (Default + 9 Orientierungen) mit Labels:
    - z = Default/Fallback
    - a = TopLeft, b = Top, c = TopRight
    - d = Left, e = Middle, f = Right
    - g = BottomLeft, h = Bottom, i = BottomRight
  - Drag & Drop für Bilder (PNG, JPG, JPEG, GIF)
  - Live-Thumbnails (120x120px skaliert)
  - Auto-Save nach jedem Drop
  - Buttons:
    - Open Resources - Öffnet resources/images/itemsOrientation/
    - Clear All - Löscht alle Orientation-Images
    - Save - Manuelles Speichern
    - Close - Editor schließen

  🎮 Verwendung:

  1. Im Editor:
    - Szene laden
    - Button "Open Orientation Editor" klicken
    - Item aus ComboBox wählen
    - Bilder per Drag & Drop auf Felder ziehen
    - Automatisches Speichern
  2. Im Spiel:
    - Item-Checkbox "Follow" oder "Click" aktivieren
    - Maus bewegen
    - Console zeigt: "Item 'player' Orientation: TopRight"
    - Bild wechselt automatisch
  3. Empfohlene Bildorganisation:
  resources/images/itemsOrientation/
    ├─ player_topLeft.png
    ├─ player_top.png
    ├─ player_topRight.png
    ├─ player_left.png
    ├─ player_middle.png
    ├─ player_right.png
    ├─ player_bottomLeft.png
    ├─ player_bottom.png
    └─ player_bottomRight.png

  ✨ Besonderheiten:

  - Performance-optimiert: Nur Items mit isFollowingMouse/Click werden aktualisiert
  - Spam-Prevention: Debug-Log nur bei Orientierungswechsel
  - Fallback-System: Kein Orientation-Image? → Verwendet Conditional/Default Image
  - Middle-Zone: Cursor innerhalb 20px vom Item-Zentrum = "Middle"
  - Auto-Save: Jede Änderung wird sofort gespeichert

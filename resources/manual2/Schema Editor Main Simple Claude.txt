│ BEHAVIOR & FEATURES                                                                                                                                                                               │
│ #############################################################################                                                                                                                     │
│                                                                                                                                                                                                   │
│ 1. AUTO-SAVE                                                                                                                                                                                      │
│    Every change auto-saves immediately:                                                                                                                                                           │
│    - Item property edits → resources/items/<item>.txt                                                                                                                                             │
│    - Item position/size drag → resources/items/<item>.txt                                                                                                                                         │
│    - Point drag → resources/items/<item>.txt                                                                                                                                                      │
│    - Scene changes → resources/scenes/<scene>/<subscene>.txt                                                                                                                                      │
│    - Condition changes → resources/conditions/conditions.txt                                                                                                                                      │
│                                                                                                                                                                                                   │
│    Save pipeline:                                                                                                                                                                                 │
│    UI Change → Object Update → Saver.save() → File → Log confirmation                                                                                                                             │
│                                                                                                                                                                                                   │
│ 2. LIVE UPDATE                                                                                                                                                                                    │
│    All changes are immediately reflected:                                                                                                                                                         │
│    - Edit item property → Repaint game window                                                                                                                                                     │
│    - Drag item → Update position → Save → Repaint                                                                                                                                                 │
│    - Toggle Show checkbox → Hide/show in game window                                                                                                                                              │
│                                                                                                                                                                                                   │
│ 3. EDIT MODE vs GAME MODE                                                                                                                                                                         │
│    Toggle with ALT+S or [Toggle Edit Mode] button                                                                                                                                                 │
│                                                                                                                                                                                                   │
│    EDIT MODE (showPaths = true):                                                                                                                                                                  │
│    - Items are draggable                                                                                                                                                                          │
│    - Items have visible bounds (cyan rectangle)                                                                                                                                                   │
│    - Corner handles for resizing (green squares)                                                                                                                                                  │
│    - Points are visible (colored circles)                                                                                                                                                         │
│    - Points are draggable                                                                                                                                                                         │
│    - Show checkboxes control visibility                                                                                                                                                           │
│    - Semi-transparent items (50% opacity)                                                                                                                                                         │
│                                                                                                                                                                                                   │
│    GAME MODE (showPaths = false):                                                                                                                                                                 │
│    - Items are NOT draggable                                                                                                                                                                      │
│    - No visible bounds/handles                                                                                                                                                                    │
│    - Points are hidden                                                                                                                                                                            │
│    - Full opacity items                                                                                                                                                                           │
│    - Normal click actions work                                                                                                                                                                    │
│                                                                                                                                                                                                   │
│ 4. SHOW CHECKBOXES                                                                                                                                                                                │
│    Control visibility in EDIT MODE only:                                                                                                                                                          │
│                                                                                                                                                                                                   │
│    Item Properties Panel:                                                                                                                                                                         │
│    - Image Show [x]: Hide/show item image                                                                                                                                                         │
│    - CustomClickArea Show [x]: Hide/show orange points                                                                                                                                            │
│    - MovingRange Show [x]: Hide/show green points                                                                                                                                                 │
│    - Path Show [x]: Hide/show red points                                                                                                                                                          │
│                                                                                                                                                                                                   │
│    ItemManagerDialog:                                                                                                                                                                             │
│    - Per-item Show   [x]: Hide entire item in editor        Purpose: Avoid clutter when editing overlapping items                                                                                                             │
│    - Per-item Follow [x]: Sets the "isFollowingMouse" boolean Value in the "<nameOfSubScene>.txt resource file  
     - Per-item Click  [x]: Sets the "isFollowingOnMouseClick" boolean Value in the "<nameOfSubScene>.txt resource file                                                                                                                                                                                       │
│                                                                                                                                                                                                   │
│                                                                                                                                                                                                   │
│ 5. SPLIT PANE PERSISTENCE                                                                                                                                                                         │
│    Divider positions are saved to EditorSettings:                                                                                                                                                 │
│    - Main split (left editors vs right game window)                                                                                                                                               │
│    - Items split (item list vs properties panel)                                                                                                                                                  │
│                                                                                                                                                                                                   │
│    Restored on next editor launch                                                                                                                                                                 │
│                                                                                                                                                                                                   │
│ 6. WINDOW MANAGEMENT                                                                                                                                                                              │
│    When opening Scene Point Editor:                                                                                                                                                               │
│    - Main Editor is minimized                                                                                                                                                                     │
│    - Only Scene Point Editor + Game Window visible                                                                                                                                                │
│                                                                                                                                                                                                   │
│    When closing Scene Point Editor:                                                                                                                                                               │
│    - Main Editor is restored                                                                                                                                                                      │
│                                                                                                                                                                                                   │
│ 7. SCENE LOADING                                                                                                                                                                                  │
│    When clicking on SubScene in tree:                                                                                                                                                             │
│    - Load SubScene file                                                                                                                                                                           │
│    - Parse conditions, image, items                                                                                                                                                               │
│    - Load all items from resources/items/                                                                                                                                                         │
│    - Display background image                                                                                                                                                                     │
│    - Render all items                                                                                                                                                                             │
│    - Update Items list                                                                                                                                                                            │

│ LOADING SCENES:                                                                                                                                                                                   │
│ ---------------                                                                                                                                                                                   │
│ User clicks SubScene in tree                                                                                                                                                                      │
│   └→ FileHandlingSimple.loadSubScene(sceneName, subSceneName)                                                                                                                                     │
│       └→ Parse .txt file (conditions, image, items)                                                                                                                                               │
│           └→ For each item name:                                                                                                                                                                  │
│               └→ ItemLoader.loadItemByName(itemName)                                                                                                                                              │
│                   └→ Parse resources/items/<itemName>.txt                                                                                                                                         │
│                       └→ Create Item object with all properties                                                                                                                                   │
│                           └→ scene.addItem(item)                                                                                                                                                  │
│                               └→ game.setCurrentScene(scene)                                                                                                                                      │
│                                   └→ game.repaint()                                                                                                                                               │
│                                                                                                                                                                                                   │
│ SAVING ITEMS:                                                                                                                                                                                     │
│ -------------                                                                                                                                                                                     │
│ User edits property in UI                                                                                                                                                                         │
│   └→ Event listener triggered                                                                                                                                                                     │
│       └→ item.setProperty(newValue)                                                                                                                                                               │
│           └→ ItemSaver.saveItemByName(item)                                                                                                                                                       │
│               └→ Write to resources/items/<itemName>.txt                                                                                                                                          │
│                   └→ Log: "✓ Auto-saved <itemName>"                                                                                                                                               │
│                       └→ game.repaint()                                                                                                                                                           │
│                                                                                                                                                                                                   │
│ DRAGGING ITEMS:                                                                                                                                                                                   │
│ ---------------                                                                                                                                                                                   │
│ User drags item in game window                                                                                                                                                                    │
│   └→ AdventureGame.handleItemDrag(dragPoint)                                                                                                                                                      │
│       └→ draggedItem.setPosition(newX, newY)                                                                                                                                                      │
│           └→ game.repaint() (live update)                                                                                                                                                         │
│               └→ On mouse release:                                                                                                                                                                │
│                   └→ AdventureGame.handleItemRelease()                                                                                                                                            │
│                       └→ ItemSaver.saveItemByName(draggedItem)                                                                                                                                    │
│                           └→ Log: "✓ Auto-saved item to resources/items/<item>.txt"                                                                                                               │
│                               └→ SceneSaver.saveScene(currentScene)                                                                                                                               │
│                                                                                                                                                                                                   │
│ DRAGGING POINTS:                                                                                                                                                                                  │
│ ----------------                                                                                                                                                                                  │
│ User drags point in game window                                                                                                                                                                   │
│   └→ AdventureGame.handlePathPointDrag(dragPoint)                                                                                                                                                 │
│       └→ selectedPathPoint.x = dragPoint.x                                                                                                                                                        │
│           └→ selectedPathPoint.y = dragPoint.y                                                                                                                                                    │
│               └→ scenePointEditor.updatePointInList(index, x, y, type)                                                                                                                            │
│                   └→ Update point panel fields (live update)                                                                                                                                      │
│                       └→ game.repaint() (show new position)                                                                                                                                       │
│                           └→ On mouse release:                                                                                                                                                    │
│                               └→ AdventureGame.handlePathPointRelease()                                                                                                                           │
│                                   └→ ItemSaver.saveItemByName(selectedItemForPointDrag)                                                                                                           │
│                                       └→ Log: "✓ Auto-saved point to resources/items/<item>.txt"                                                                                                  │
│                                                                                                                                                                                                   │
│ ADDING ITEMS TO SUBSCENE:                                                                                                                                                                         │
│ --------------------------                                                                                                                                                                        │
│ User clicks [Add Item to Scene]                                                                                                                                                                   │
│   └→ Open ItemManagerDialog                                                                                                                                                                       │
│       └→ User selects item and clicks [Add to Scene]                                                                                                                                              │
│           └→ ItemLoader.loadItemFromDefault(itemName)                                                                                                                                             │
│               └→ currentSubScene.addItem(item)                                                                                                                                                    │
│                   └→ FileHandlingSimple.saveSubSceneToDefault(subScene, sceneName)                                                                                                                │
│                       └→ Log: "✓ Added item to SubScene"                                                                                                                                          │
│                           └→ loadItemsIntoEditor() (refresh item list)                                                                                                                            │
│                               └→ game.repaint()                                                                                                                                                   │

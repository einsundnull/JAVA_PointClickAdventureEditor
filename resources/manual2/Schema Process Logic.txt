│                                                                                                                                                                                                   │
│ OVERVIEW:                                                                                                                                                                                         │
│ Processes are actions that execute when an Action (give, take, use, etc.)                                                                                                                         │
│ is applied to a CustomClickArea of an Item.                                                                                                                                                       │
│                                                                                                                                                                                                   │
│ EXECUTION FLOW:                                                                                                                                                                                   │
│ 1. User clicks on Item with selected Action                                                                                                                                                       │
│ 2. IF Conditions are checked                                                                                                                                                                      │
│ 3. If all conditions are met → THEN block executes                                                                                                                                                │
│ 4. Actions execute sequentially in order                                                                                                                                                          │
│ 5. Process completes                                                                                                                                                                              │
│                                                                                                                                                                                                   │
│ ═══════════════════════════════════════════════════════════════════════════════                                                                                                                   │
│ FILE STRUCTURE                                                                                                                                                                                    │
│ ═══════════════════════════════════════════════════════════════════════════════                                                                                                                   │
│                                                                                                                                                                                                   │
│ Location: resources/processes/<ItemName>_<ActionName>.txt                                                                                                                                         │
│ Example:  resources/processes/cup_take.txt                                                                                                                                                        │
│           resources/processes/door_use_key.txt                                                                                                                                                    │
│                                                                                                                                                                                                   │
